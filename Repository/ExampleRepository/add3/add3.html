Write a static method add1 that takes in an int and returns the int three higher

The table below shows various calls and the values they should return:

Call Value Returned Reason
add1(2); 5 2 + 3 = 5
add1(-1) 2 -1 + 3 = 4
add1(1000) 1003 1000 + 3 = 1003

You may define private helper methods to solve this problem, but otherwise you may not call any other methods of the class nor create any data structures such as arrays, lists, etc.
